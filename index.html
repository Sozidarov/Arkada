ArkadaTool

OSINT Инструмент для поиска информации о людях по базам данных.

ArkadaRAT

Вирусное ПО для дистанционного управления компьютером пользователя. Управление через Telegram Бота.

ArkadaLOCK

Позволяет заблокировать компьютер человеку, код приходит вам в Telegram Bot. Принцип работы WinLocker.

SPUTNIK (Разработка)

OSINT Инструмент для сети и сайтов, имеется parser, ip probiv, dos...
